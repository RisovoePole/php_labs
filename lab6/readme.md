### Шаг 1

Определим модель данных, которую будем хранить:

- `title` - название поста
- `author` - имя автора (его user name)
- `tags` - массив из **enum** - то, о чём идёт речь в посте
- `body` - тело поста - основной текст
- `like`
- `dislike`
- `createdAt` - авто заполняющаяся дата - когда пост был отправлен.
